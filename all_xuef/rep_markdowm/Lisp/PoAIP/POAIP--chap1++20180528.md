## 1.6 using function

```lisp
> (mapcar #'- '(1 2 3 4))
(-1 -2 -3 -4)

> (mapcar #'+ '(1 2 3 4) '(10 20 30 40))
(11 22 33 44)
```



## 1.7 Higher-Order  Functions

```lisp
> (apply #'+ '(1 2 3 4))
10
```

```lisp
[4]> (defun self-and-double (x) (list x (+ x x)))
SELF-AND-DOUBLE
[5]> (self-and-double 4)
(4 8)

[6]> (mapcar #'self-and-double '(1 3 100))
((1 2) (3 6) (100 200))
```



```lisp
> (funcall #'+ 1 2 3 4)
10
```
