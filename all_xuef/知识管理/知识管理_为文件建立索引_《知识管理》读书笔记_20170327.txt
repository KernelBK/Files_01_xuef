知识为何没了力量之一——碎片化
之前想就[知识为何没了力量]这个问题写篇东西，但觉得问题太大了，我把握不了，就没写。或许可以一点点写。
今天的主题是——如何克服知识的碎片化
网络时代的一个显著特征就是：信息爆炸。我们穿梭于信息洪流中，我们好像什么都知道，但实际上什么也不知道。
孤立地知识点从迁移应用的角度来说是没有多大意义的，这是知识为何没了力量的一个原因。我们有的只是孤立的零碎的知识！
什么样的知识才有力量？ 体系化的知识，经过你彻底理解消化后的知识。
怎么将零散的知识串起来从而内化，使之真正属于我们自己的。一个很有用的方法就是——为你的知识建立索引
科学地建立索引后，便能条理化保存，随时快速调用。而且在这个建立索引的过程中，你加深了对该知识的理解。

下面说下具体操作： 
2016年8月我读了关于知识管理的一本小书，第一次认真接触“知识管理”这个概念。（概念是最厉害的抽象工具，我们要注意）
书中强烈建议我们要条理化保存知识，这是构建体系化知识的重要一步。
如何做呢？很简单，那就是——合理命名你的文件
对于我来说，有电子书，自己的读书笔记，收集的好文章，自己写的文章，心得感想日记，个人财务记录...
 
对于这些文件，你要有个统一的命名原则： （我是这样给文件命名的）
a 对于图书：书名 + 作者 + 出版时间 + 相遇时间 + 类别 + 主题 + 关键词
b 对于读书笔记：读书笔记 + 书名 + Dongxuef + 时间 + 主题 + 关键词
c 对于好文章：作者 + 文章名 + 写作时间 + 相遇时间 + 类别 + 主题 + 关键词
d 对于每日记录：类别 + 日期 + 关键词 + 备注

注意：后面的主题和关键词很重要，它反映了你消化知识的过程，并且在你将来写文章时可以随时根据主题调用。

最后，向大家推荐一款软件：everything 
它通过建立高效索引的方式，快速搜索你电脑中的文件，速度比系统自带的搜索快千倍万倍不止。
比如你想查找关于“知识管理”的内容，只要输入关键词或主题“知识管理”，瞬间就可以弹出。

好了，快为你的文件或者朋友圈、空间的文章梳理下吧。
走出构建体系化知识的第一步。
