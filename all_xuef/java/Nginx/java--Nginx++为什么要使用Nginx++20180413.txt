
## nginx
反向代理 负载均衡 动静分离
core：分布式部署、应用的垂直水平切分，为应用系统向大规模架构变迁做好准备。

虽然动静分离能提升一点性能，但Nginx的主要关注点不是直接提升性能，而是使得系统的分布式，大规模部署成为可能。
在小系统上应用Nginx，可能还会导致性能的下降，毕竟多了一次代理，多了在Nginx和应用服务器之间的交互和网络传输。

## 思考
如果访问量持续增大，体系结构如何演变
