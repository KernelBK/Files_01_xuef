## 查看命令可携带参数
dir /?


cls 清屏
md 建立子目录
cd 进入目录
	cd /
	cd \ 回到根目录
	cd ..
	cd \
rd 删除子目录(目录必须为空) 
	
dir 查看目录下文件
	dir /ON 按名字排序

tree 显示磁盘目录结构

delete 删除整个目录

### 环境变量
通过echo %变量名% 查看环境变量值，
echo %path%
echo %JAVA_HOME%
set 变量名查看环境变量名  或者set 变量名=新值   设置新值（当前窗口有效）

### 启动，关闭服务
net start mysql
net stop mysql
(cmd 以管理员身份启动)


### 更改时间
打开自己的点脑，打开运行，用win键+R可快速打开
在输入框输入cmd，点击确定，进入dos界面
输入time点击自己电脑键盘上回车，现在输入现在的时间即可更改，格式和上文的格式一样，如14:12:30 
完成后点击回车即可修改成功 注意输入法调成英文

输入date即可更改日期和更改时间方法基本一样，如2015/5/6，完成后点击回车即可修改成功，注意：输入法调成英文，
