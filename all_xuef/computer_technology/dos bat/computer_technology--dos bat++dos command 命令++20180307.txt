## 查看命令可携带参数
dir /?


cls 清屏
md 建立子目录
cd 进入目录
	cd /
	cd \ 回到根目录
	cd ..
	cd \
rd 删除子目录(目录必须为空) 
	
dir 查看目录下文件
	dir /ON 按名字排序

tree 显示磁盘目录结构

delete 删除整个目录

### 环境变量
通过echo %变量名% 查看环境变量值，
echo %path%
echo %JAVA_HOME%
set 变量名查看环境变量名  或者set 变量名=新值   设置新值（当前窗口有效）

### 启动，关闭服务
net start mysql
net stop mysql
(cmd 以管理员身份启动)
